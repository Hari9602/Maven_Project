1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/simple-calculator.git
   ```
2. Navigate into the project directory:
   ```bash
   cd simple-calculator
   ```

3. Build the project using Maven:
   ```bash
   mvn clean install
   ```
